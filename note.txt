#2.5
* 바벨 설치
npm install @babel/node
* 커맨드 라인 입력:
npm install @babel/preset-env
* 커맨드 라인 입력:
npm install @babel/core
* 커맨드 라인 입력:
npm install core-js@3

* .babelrc 파일을 만들어주세요
* .babelrc 파일에 아래 내용을 입력 후 저장해주세요:
{
    "presets": [
        [
            "@babel/preset-env",
            {
                "useBuiltIns": "entry",
                "corejs":3
            }
        ]
    ]
}
